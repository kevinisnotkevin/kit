Parrot Security OS (или ParrotSec) —дистрибутив Linux, основан на Debian с упором на компьютерную безопасность. Предназначен для тестирования системы на проникновение, оценки уязвимостей и ликвидации их последствий, компьютерной криминалистики и анонимного просмотра веб-страниц. Разработан командой Frozenbox.

В качестве графического окружения используется среда рабочего стола MATE.

Parrot Linux включает в себя готовый набор инструментов для разработки приложений. В системе предустановлены различные фреймворки, библиотеки, компиляторы, дизассемблеры, IDE и редакторы кода.

Криптографическое ПО дистрибутива позволяет полностью защитить ваши данные. Всю систему можно установить в полностью зашифрованный раздел на диске, что обеспечит безопасность данных в случае кражи компьютера.

Специальная программа AnonSurf позволяет запускать TOR и «пропускать» весь интернет трафик через TOR. Также есть возможность использовать DNS сервера, отличные от тех, которые предоставлены провайдером.

Еще один пример использования Parrot Linux в качестве мониторинга уязвимостей - сканер уязвимостей виртуальной телефонной станции “Rtpflood”. Простой, но в то же время мощный инструмент для перебора корпоративной директории VoIP, который имитирует поведение IP телефона чтобы загрузить имя и расширенные записи, которые данный телефон может отобразить на своём экране.
