# vim

```
vimtutor - туториал

[ESC]    - В нормальный режим

h        - Влево
j        - Вниз
k        - Вверх
l        - Вправо

i        - Режим редактирования
a        - Режим добавления
x        - Удалить символ под курсором

:q!      - Выйти без сохранения
:wq      - Выйти с сохранением

dw       - Удалить текст от курсора до конца слова без пробела
d2w      - Удалить два слова после курсора
de       - Удалить текст от курсора до конца слова с пробелом
d$       - Удалить все, что справа от курсора
dd       - Удалить строку
2dd      - Удалить две строки

0        - Переместить курсор на начало строки
2w       - Переместить курсор на начало второго слова после курсора
3e       - Переместить курсор на конец третьего слова после курсора

u        - Отменить одно изменение строки
U        - Отменить все изменения строки
[CTRL+R] - Отменить отмену
```
