### –ò—Å–∫–ª—é—á–µ–Ω–∏—è

Python –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –∏—Å–∫–ª—é—á–µ–Ω–∏–µ –≤—Å—è–∫–∏–π —Ä–∞–∑, –∫–æ–≥–¥–∞ –¥–µ–ª–∞–µ—Ç—Å—è –ø–æ–ø—ã—Ç–∫–∞ –≤—ã–ø–æ–ª–Ω–∏—Ç—å –Ω–µ–¥–æ–ø—É—Å—Ç–∏–º—ã–π –∫–æ–¥. –ú—ã —É–∂–µ –∑–Ω–∞–µ–º –æ —Ç–æ–º, –∫–∞–∫ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—Ç—å –∏—Å–∫–ª—é—á–µ–Ω–∏—è Python —Å –ø–æ–º–æ—â—å—é –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π `try` –∏ `except`, –ø–æ–∑–≤–æ–ª—è—é—â–∏—Ö –∏–∑–±–µ–∂–∞—Ç—å –ø—Ä–µ–∂–¥–µ–≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ –ø—Ä–µ–∫—Ä–∞—â–µ–Ω–∏—è —Ä–∞–±–æ—Ç—ã –ø—Ä–æ–≥—Ä–∞–º–º—ã –ø—Ä–∏ –≤–æ–∑–Ω–∏–∫–Ω–æ–≤–µ–Ω–∏–∏ –ø—Ä–æ–±–ª–µ–º–Ω—ã—Ö —Å–∏—Ç—É–∞—Ü–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã –ø—Ä–µ–¥–≤–∏–¥–µ–ª–∏. –ù–æ –≤ –ø—Ä–æ–≥—Ä–∞–º–º–µ –º–æ–∂–Ω–æ –≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞—Ç—å –∏ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –∏—Å–∫–ª—é—á–µ–Ω–∏—è. –ü–æ—è–≤–ª–µ–Ω–∏–µ –∏—Å–∫–ª—é—á–µ–Ω–∏—è —Ä–∞–≤–Ω–æ—Å–∏–ª—å–Ω–æ —Å–ª–µ–¥—É—é—â–µ–º—É –ø—Ä–∏–∫–∞–∑—É: ‚Äú–ü—Ä–µ–∫—Ä–∞—Ç–∏ –≤—ã–ø–æ–ª–Ω—è—Ç—å –∫–æ–¥ –¥–∞–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ –∏ –ø–µ—Ä–µ–π–¥–∏ –∫ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—é –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ `except`‚Äù.

### –ò—Ç–µ—Ä–∞—Ç–æ—Ä

–û–±—ä–µ–∫—Ç-–∏—Ç–µ—Ä–∞—Ç–æ—Ä - –æ–±—å–µ–∫—Ç, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π –ø–æ—Ç–æ–∫ –¥–∞–Ω–Ω—ã—Ö. –ò—Ç–µ—Ä–∞—Ç–æ—Ä –¥–æ–ª–∂–µ–Ω —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –¥–≤–∞ –º–µ—Ç–æ–¥–∞: `__iter__()` - –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –æ–±—å–µ–∫—Ç –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞; `__next__()` - –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞ –∏–ª–∏ `StopIteration`, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –Ω–µ—Ç.
```python
class SimpleIterator:
    def __init__(self, start, end):
        self.current = start
        self.end = end

    def __iter__(self):
        return self

    def __next__(self):
        if self.current < self.end:
            result = self.current
            self.current += 1
            return result
        else:
            raise StopIteration

my_iterator = SimpleIterator(1, 5)

for value in my_iterator:
    print(value)
```
